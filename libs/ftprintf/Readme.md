# ft_printf project

(Origin is at:)[https://github.com:pvolikov42/ft_printf.git]
> Currently private because of learning purposes

*Made for training/learning purposes*

**ft_printf** is a self-made analog of printf (stdio.h) function.
Currently only works with %d %i %s %x %X %c %p %%
Other converters are not implemented. Flags aren't implemented.

Relies on libft
Update LIBFT_DIR in Makefile according to location of libft

`make`

`make test`
